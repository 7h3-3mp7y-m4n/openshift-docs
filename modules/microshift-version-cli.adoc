= Checking the {product-title} Version from the command-line

.Procedure

* Run the following command to check the version information:

[source,terminal]
----
$ microshift version
----

.Example output
[source,terminal]
----
MicroShift Version: 4.10.0-0.microshift-e6980e25
Base OCP Version: 4.10.18
----
